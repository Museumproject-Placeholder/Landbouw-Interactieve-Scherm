# Museum Landbouw Project

# Prerequisites
- nodejs (v22.19.0) met npm
- React
- Tailwindcss (v3.4.17)
- VSCode (ES7+ React/Redux/React-Native snippets, Tailwind CSS IntelliSense, Auto Rename Tag)
- Git Desktop [guide](github_desktop_ssh_setup.md)
